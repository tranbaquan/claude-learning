# Skill Setup

## Playwright CLI
> Note: Playwright CLI Skills are the recommended way to use Playwright with Coding Agents CLI (Ex: Claude).
> You can ignore Playwright MCP setup instructions if you are using Playwright CLI Skills and vice versa.

Install the latest Playwright CLI globally:
```shell
npm install -g @playwright/cli@latest
```

Then, install the Playwright CLI skill for Claude:
```shell
playwright-cli install --skills
```
